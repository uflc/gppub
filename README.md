주요 특징

스테이지가 절차적으로 난이도, 팩션, 타일셋 따라 구성되어져 한 스테이지에서도 다양한 플레이 경험이 가능,

미션 성공 보상이나 미션 중의 루팅(파밍)을 통해 더 좋은 무기나 스킬을 갖춰가며 최종 목표까지 도전.

진행될 수록 난이도는 더욱 어려워짐.

로드 아웃 레벨에서 미션에 들고갈 여러 스킬, 장비, 소모품 등은 미션 도중 바꿀수 없으므로 잘 선택해야 함.
